Feeds & data

Every view here is also a machine-readable feed — add a format suffix to any table URL:

  • /locations/current.json?_shape=objects — current records as JSON
  • /locations/current.geojson — current records as a GeoJSON FeatureCollection
  • /locations/archived.json?_shape=objects — archived records as JSON
  • /locations/archived.geojson — archived records as GeoJSON

CORS is enabled, so the feeds can be used directly from the browser — they power the maps in The Tampa Monitor's weekly agenda coverage. GeoJSON output via datasette-geojson.

About this data

Source: City of Tampa Open Data Portal — Development Coordination layer.

An automated pipeline fetches the City's ArcGIS feed nightly at 00:00 UTC, diffs it against the previous run, and moves any record that drops off the feed into the archive with a timestamp.
